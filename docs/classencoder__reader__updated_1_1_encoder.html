Detailed Description

This class implements an encoder for the motor for an ME405 kit.

The current motor used, a Pololu 50:1 motor, has 64 ticks per revolution internally and a 50:1 output ratio, giving the motor 3200 ticks per revolution. For a different motor, the value self.autoreload must be changed to the appropriate ticks per revolution.
